# Personal Finance Manager
## 📖 Description
The **Personal Finance Manager** is a web-based application developed using PHP, MySQL, HTML, CSS, and JavaScript. It helps users efficiently manage their daily financial activities by tracking income, expenses, and budgets in an organized way.

The system provides a user-friendly interface where users can securely register, log in, and maintain their financial records. It allows users to analyze their financial data and improve spending habits.

---

## 🚀 Features

- 🔐 User Authentication  
  - Register  
  - Login  
  - Forgot Password  
  - Reset Password  

- 💰 Income Management  
  - Add Income  
  - View Income  

- 💸 Expense Management  
  - Add Expense  
  - View Expense  
  - Search Expense  
  - Download Expense List as PDF  

- 📊 Budget Management  
  - Set Budget  
  - View Budget  

- 📈 Reports  
  - Monthly Reports  
  - Category-wise Reports  

- 🗂️ Category Management  

- 📉 Data Visualization using Charts
